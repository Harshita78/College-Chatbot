#  CampusIntel – College Enquiry Chatbot

CampusIntel is an AI-powered college enquiry chatbot designed to answer student queries related to admissions, courses, fees, placements, campus facilities, and more.

It uses Natural Language Processing (NLP) and a PyTorch-based intent classification model to understand user queries and generate relevant responses.

---

## Tech Stack

- Python
- PyTorch
- NLTK (Tokenization & Lemmatization)
- Flask (for deployment)
- JSON (for intent storage)

---

##  How to Run

1. Clone the repository
git clone https://github.com/Harshita78/College-Chatbot.git

2. Install dependencies
pip install -r requirements.txt

3. Train the model
python train.py

4. Run chatbot
python chat.py
python app.py
